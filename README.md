# 🎮 Tetris Clone on Phaser

Простий клон класичного **Тетрісу**, створений на базі бібліотеки **Phaser 3**.  
Гра підтримує керування як з клавіатури, так і з сенсорних пристроїв.

---

## 🕹️ Демонстрація
[🎯 Live Demo](https://tetris-phaser.vercel.app/)  

---

## 🚀 Функціонал
- Рух фігур клавішами або свайпами 📱  
- Обертання фігур 🔄  
- Автоматичне падіння ⬇️  
- Визначення заповнених ліній 💥  
- Підрахунок очок 💯  
- Фонова музика 🎵  
- Підтримка мобільних пристроїв  

---

## 🧩 Технології
- **Phaser 3**
- **JavaScript (ES6)**
- **Vite** — для локального запуску
- **HTML / CSS**

---

## 🕹️ Управління

**На комп’ютері:**
- ⬅️ / ➡️ — рух фігури  
- ⬆️ — обертання  
- ⬇️ — прискорення  

**На мобільному:**
- Свайп вліво / вправо — рух фігури  
- Тап — обертання
